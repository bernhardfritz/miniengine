Function smoothstep

  • smoothstep<T>(edge0, edge1, x): WidenGenType<T>

  • Perform Hermite interpolation between two values.

    Type Parameters

    • T extends genType

    Parameters

    • edge0: T

      Specifies the value of the lower edge of the Hermite function.

    • edge1: T

      Specifies the value of the upper edge of the Hermite function.

    • x: T

      Specifies the source value for interpolation.

    Returns WidenGenType<T>

    Performs smooth Hermite interpolation between 0 and 1 when edge0 < x < edge1. This is useful in cases where a threshold function with a smooth transition is desired. Returns 0.0 if x ≤ edge0 and 1.0 if x ≥ edge1.
